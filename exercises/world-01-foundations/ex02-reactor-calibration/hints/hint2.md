## Hint 2: Tools & Types

- **Integer arithmetic:** `+`, `-`, `*`, `/` work on i32 and u8.
- **Type coercion:** When mixing i32 and u8, think about operator precedence and which type the result will be.
- **Division:** Integer division truncates; `5 / 2 = 2`, not 2.5.
- **Let bindings:** `let result = ...;` creates a variable.
- **Return:** The last expression in a function is automatically returned.
