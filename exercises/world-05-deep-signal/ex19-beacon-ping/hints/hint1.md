## Hint 1: Conceptual Question

You need to make an HTTP GET request and return the URL, status code, and response body. Before looking up the API: what information do you need to assemble the URL? Once you have a response, what steps do you need to take to get the status code and the text body? Think about what could go wrong at each step.

**Spoiler threshold:** Low—asks you to plan the shape of the solution.
