## Hint 1: Conceptual Question

This capstone has three functions. Before writing any code, map out the pipeline: what does each function need as input? What does each produce? How does `run_deep_signal_pipeline` delegate to the other two? For `build_deep_signal_report`, how do you split a collection into two groups and compute a mean—without a network call?

**Spoiler threshold:** Low—asks you to design the pipeline before implementing it.
